## [0.2.0] - 2025-12-12

### Added

- Comprehensive SECURITY.md documenting security architecture, device provisioning, certificate lifecycle, and compliance (POPIA/GDPR)
- Complete Smart Plug AI plan.txt as canonical project plan with security-enhanced phases and technical implementation details
- CHANGELOG.md for tracking project changes

### Changed

- Updated README.md to provide concise project summary with links to detailed documentation
- Restructured documentation to emphasize security-first architecture
- Enhanced project plan with detailed security architecture and implementation roadmap

### Documentation

- Added security architecture documentation covering device, communication, data, and access control security
- Added device hardening checklist for firmware, network, and physical security
- Added vulnerability disclosure policy with clear reporting procedures and response timelines
- Documented POPIA and GDPR compliance measures
- Updated README to focus on quick start and contribution guidelines

## [0.1.0] - 2024-12-XX

### Initial Release

- Initial project structure with mobile app, web dashboard, firmware, and backend directories
- Comprehensive documentation including ARCHITECTURE.md, API.md, ROADMAP.md
- Soldering checklist for hardware preparation
- CODE_OF_CONDUCT.md and CONTRIBUTING.md for community guidelines
- MIT License

### Project Documentation

- System architecture documentation with component diagrams
- API documentation for REST, WebSocket, and MQTT interfaces
- Project roadmap with phase-by-phase implementation plan
- Hardware soldering preparation guide
